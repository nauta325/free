🤖 직접 만들고 싶다면 (보너스)
아래를 Claude(claude.ai)에 붙여넣으세요. Claude는 결과를 A4 2단 PDF로 바로 뽑아줘서 그대로 인쇄해 쓸 수 있어요. (다른 AI는 PDF 출력이 깔끔하게 안 나와요.) 
너는 중학교 영어 내신 전문 강사야.
"수동태" 단원에서 학생들이 자주 틀리는
함정 위주로 연습 문제 50개를 만들어줘.

[조건]
1. 난이도는 기초 / 표준 / 고난도로 표시.
2. 문제 유형을 골고루 섞어줘:
   - 부정문·의문문 전환
   - 어법상 올바른 형태 고르기 (빈칸 + 동사원형 제시)
   - 틀린 곳 찾아 고치기
   - 의미 차이로 답이 갈리는 함정 문제
3. 각 문제 위에 [부정문으로], [틀린 곳 고치기]처럼
   무엇을 하라는지 한글로 표시.
4. 정답은 본문에 섞지 말고 맨 끝에 따로 모아줘.
5. A4 2단으로 PDF 출력이 가능하게 정리해줘.
💡 팁. "수동태" 자리를 다른 단원(시제·관계대명사 등)으로 바꾸면 어느 단원이든 똑같이 만들어져요. 만든 뒤 "PDF로 만들어줘"라고 하면 바로 출력돼요.
